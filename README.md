# Tenki Japan Sales Search

Static GitHub Pages dashboard for searching Tenki Japan daily sales by product genre, shop, year, month, and day.

## Run Locally

```bash
python3 -m http.server 8766
```

Then open `http://localhost:8766`.

## Data

The `data/` folder contains generated daily sales CSV files derived from the Tenki Parquet files. Do not publish this repository publicly unless the sales data is approved for public access.
